# Tony Design Construction LLC - Frontend

Sitio web oficial de Tony Design Construction LLC.

- Página principal (`index.html`)
- Todos los proyectos (`all-projects.html`)

## Tecnologías

- CSS3
- JavaScript (vanilla)
- Backend conectado a FastAPI en Render
